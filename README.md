Focus List

A simple and efficient task management app built with Flutter, designed to help users stay focused and organized. This project is part of my portfolio and demonstrates my skills in Flutter development, state management, and local data persistence using Hive.

🚀 Features

Add, edit, delete tasks with ease.

Mark tasks as done using swipe gestures.

Undo delete with SnackBar action.

Reorder tasks with drag-and-drop.

Persistent storage using Hive for offline support.

Clean UI with focus on usability.

🛠️ Tech Stack

Flutter → Cross-platform UI toolkit.

Hive → Lightweight and fast NoSQL database for local persistence.

Dart → Programming language powering Flutter.
